# Version History

| Version | Changes |
| :--- | :--- |
| **Version 1.4.0** | **NEW FEATURES**<br>- **Added Character Portrait** Added the option to include the character's portrait. Technically, this converts the image into Base64 code and includes it in the output HTML file so will increase the file size of the HTML file by the size of the image plus ~30%.<br>- **Added Character Information section** A new section has been added that can include the character portrait and any appearance, identity or biographical information about the character.<br><br>Note that the Portrait and the Character Information section can be shown or hidden independently of each other.<br><br>**BUG FIXES**<br>- Prevent Sheet Exports from actors in the sidebar that do not have tokens on the scene. This is needed as the module must initialise data (attacks and spells) that can only be done if a token is available.<br>- Fixed a missing css style bug.|
| **Version 1.3.0** | **IMPROVEMENTS**<br>- **Metric Support:** Supports display of metric distances and weights.|
| **Version 1.2.0** | **NEW FEATURES**<br>- **Fully customisable exports:** You can now select your preferred layout and style separately, and turn each section on or off.<br><br>**NEW TEMPLATES/STYLES:**<br>- A **printer-friendly** style that uses less ink.<br>- A **dark mode** style.<br><br>**BUG FIXES**<br>- Fixed some missing localisation text in the templates.|
| **Version 1.1.1** | **BUG FIXES**<br>- Fixed leaking css that was forcing the Foundry settings to always show in the sidebar.|
| **Version 1.1.0** | **NEW FEATURES**<br>- **Creature Sheet support:** You can now export creature sheets as well as character sheets.<br>- **Export Preview**: See the layout and style of the sheet export in a preview window before downloading.<br>- **Multiple export templates:** You can now select from different export templates.<br>- **New Template:** Added a new Compact Template designed to look like a stat block for including in adventures, and a Fun Template for, well, fun!<br><br>**IMPROVEMENTS**<br>- **Localisation:** Added support for the exported character sheet to be localised using the current selected language set in FoundryVTT (if supported by both the RMU system and this module).<br>- Empty Skill Categories are now hidden when the sheet only shows ranked/favourite skills.|
| **Version 1.0.0** | **IMPROVEMENTS**<br>- Styled export dialogue.<br>- Replaced font awesome icons with Material Design Icons.<br><br>**BUG FIXES**<br>- Removed hardcoded "+" from resistance bonuses.|
| **Version 1.0.0-beta1** | **INITIAL RELEASE** Make sure to read the README.md first (I know you won't...you won't even be reading this).|
